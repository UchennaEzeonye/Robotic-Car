# ESP32 Robotic Car Project

## Overview

This repository contains an **ESP32-based robotic car project** that is being continuously developed and improved. The project started as a simple Bluetooth-controlled car and is gradually evolving into a more advanced and autonomous robotic system.

The focus of this project is **learning, experimentation, and progressive improvement** in robotics, embedded systems, and motor control.

---

## Project Progression

* Bluetooth control using Serial Bluetooth Monitor
* Bluetooth control using a mobile car-control app
* Integration of ultrasonic sensors for obstacle detection
* Autonomous movement without manual control (in progress)

---

## Core Features

* ESP32
* Built-in Bluetooth (Classic)
* DC motor control using H-bridge drivers
* External motor power supply
* Supports both manual and autonomous operation

---

## Technologies

* ESP32 (Arduino framework)
* BluetoothSerial
* H-bridge motor drivers (L293D)
* DC motors
* Ultrasonic sensors (ongoing integration)
